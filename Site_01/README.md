# The Clean Method

Site institucional para serviços profissionais de limpeza residencial e empresarial em Langley e região metropolitana de BC, Canadá.

## Funcionalidades

- **Site bilíngue** — suporte completo a Português e Inglês com toggle no menu
- **Design responsivo** — adaptado para desktop, tablet e mobile
- **Slider automático** — galeria de imagens de ambientes limpos (Unsplash) com transição suave
- **Formulário de contato** — com campos essenciais e validação de telefone
- **Autocomplete de endereço** — integração com a API Nominatim (OpenStreetMap) para sugerir endereços ao digitar
- **WhatsApp integrado** — botão flutuante que abre modal com formulário e envia mensagem estruturada para o WhatsApp
- **Botão voltar ao topo** — aparece ao descer a página
- **Scrollbar personalizada** — nas cores da identidade visual (roxo e cobre)
- **SEO** — meta tags, Schema.org para LocalBusiness
- **Animações de entrada** — seções aparecem suavemente ao scroll

| Roxo claro  | `#7B4FA8` | Detalhes                   |

| Cobre claro | `#D4956F` | Gradientes                 |

## Tecnologias

- HTML5 + CSS3 + JavaScript (vanilla)
- Google Fonts (Montserrat + Playfair Display)
- Unsplash (imagens de referência)
- Nominatim / OpenStreetMap (autocomplete de endereço)
- WhatsApp Click-to-Chat API

## Como usar

Abra o arquivo `index.html` diretamente no navegador. Não requer build ou dependências.
