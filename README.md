# The Clean Method

Landing pages profissionais para serviços de limpeza residencial e empresarial em Langley / BC.

## Estrutura

```
clean_method_ca/
├── Site_01/          # Landing page com visual clássico
│   ├── index.html
│   ├── logo_no_bg.png
│   ├── logo.jpeg
│   ├── vector_logo.svg
│   └── id_visual_*.jpeg
├── Site_02/          # Landing page com visual editorial
│   ├── index.html    # Minificado em 1 linha
│   └── logo_no_bg.png
└── README.md
```

## Tecnologias

- HTML5 + CSS3 (inline styles)
- JavaScript vanilla (i18n, slider, formulários)
- Google Fonts (Fraunces + Inter / Montserrat + Playfair Display)

## Funcionalidades

- **i18n** — Alternância PT/EN com persistência em localStorage
- **Slider** — Hero section com transição automática de imagens
- **Formulários** — Contato via WhatsApp com modal
- **FAQ** — Accordion com animação
- **Autocomplete** — Endereços via Nominatim (OpenStreetMap)
- **Responsivo** — Layout adaptável para desktop, tablet e mobile
